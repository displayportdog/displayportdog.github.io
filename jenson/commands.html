Here are some things that Jenson can do:

help

Display all available commands. Probably how you ended up here.

gm, ga, ge, gn

Greetings and farewells!

sprint TIME{h|m|s} [MESSAGE]

Set a timer for the duration specified, then, once the timer is up, repeat the message appended onto the timer.

whois [USER]

Get info on the tagged user!

guildinfo

Get information about the current guild!

meow/woof/screech

Cat, dog and fox pictures, for your viewing pleasure!

gif QUERY

Search for GIFs!

define/urban/thesaurise QUERY

Searches the dictionary/urban dictionary/thesaurus for your word/phrase.

lvl/leaderboard

XP related commands - check your server level, as well as the server's top members.

coinflip

Flip a coin!

choose OPTION1, OPTION2, [OPTION...]

Choose out of a list of options.

poll QUESTION | [OPTION1] | [OPTION2] | [OPTION...]

Create a poll!

yeet/yeetsleep USER

Send a user to be productive/sleep.

prompt

Get inspired!

cookie

Give a cookie!

hug

Everyone deserves love!

bday, bday-set, bday-next

Birthday related commands! Set your birthday, and see the next ten upcoming.
