Randomiser commands

A lot of Jenson's commands rely on random number generation (RNG). The issue with RNG is that it can sometimes mess up, or act more predictably than it should.

Because Jenson uses already published code, this shouldn't really be an issue. If you get the same thing twice, you probably just rolled the same thing twice.
