XP & Levelling commands

One of Jenson's primary jobs in Discordance is to keep track of user levels.

When a user types for the first time, their user ID (the long string of digits you get when you right-click and hit 'Copy ID') is added to a file beside a number.

The number is the 'XP Increment' - the amount of XP that is added onto your total score when a user speaks. To give the levelling system a sense of entropy, this number is randomly generated. By default, it is an integer ranging from 20 - 50 - however this can be altered with modifiers.

Modifiers are objects that tell Jenson how to generate the 'XP Increment'.

If users are levelling at an unusual pace, this is probably the root cause.
