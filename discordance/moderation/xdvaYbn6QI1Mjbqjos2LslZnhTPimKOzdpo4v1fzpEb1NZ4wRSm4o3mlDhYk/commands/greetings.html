Greeting commands

Jenson's code contains an array of strings - these are the prefabricated messages that are sent when a user triggers the bot via !gm, !ga, !ge or !gn. I am constantly looking for new phrases to add to these, so if you or any users have suggestions I would be eternally grateful if those were forwarded on to me.

These commands also have quite little room for error. They can be reliably used in the middle of messages, including after newlines and within words. If the commands aren't working, Jenson is probably down.

Please also note that these four commands are actually only three; !ge is an alias for !ga.


Commands used in this section

</> Brief Description Complexity
!gm Greeting command. Short for Good Morning. Easy to use; no arguments
!ga Greeting command. Short for Good Afternoon. Easy to use; no arguments
!ge Greeting command. Short for Good Evening. Easy to use; no arguments
!gn Greeting command. Short for Good Night. Easy to use; no arguments
